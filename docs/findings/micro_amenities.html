3. Machine Learning Algorithm

Finally, we apply K-means clustering to group districts based on similarities in housing prices and amenity characteristics. This unsupervised learning approach helps identify neighborhood types that share common spatial and socioeconomic features.

K-means Clustering Overview

We cluster dong-level observations using variables such as housing price, safety, transport accessibility, education environment, and green space. The algorithm groups areas with similar profiles into the same cluster.

This approach helps move beyond individual variables and reveals broader neighborhood patterns in Seoul’s housing market.

Cluster Visualization

군집 지도, scatter plot, PCA plot 등을 넣는 영역

Cluster Interpretation

The clustering results can be used to identify distinct types of neighborhoods, such as high-price / high-amenity areas, emerging mid-price districts, or lower-price neighborhoods with weaker amenity access. These groups help show how multiple factors jointly shape housing market outcomes across Seoul.
