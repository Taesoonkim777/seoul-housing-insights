Methodology

Data Collection

Apartment transaction data from MOLIT and district-level amenity data from Seoul Open Data Portal (2015–2024).

Amenity Index Construction

Composite urban amenity index using PCA and Z-score normalization across transportation, education, safety, and green space.

PCA Composite Index
03

Macro & Micro Analysis

Regression and correlation analysis examining macro drivers and micro-level amenity impacts on housing prices.

Regression Time Series

Interactive visualizations with Plotly.js and D3.js for narrative-driven storytelling.
