🧪 Research Design

Methodology & Team

Our research approach and the team behind this project.

Methodology

A structured workflow from data collection to interactive visualization.

Data Source

Data is collected from the Seoul Open Data Portal, including housing transactions and district-level amenity indicators.

Seoul Open Data

Data Processing

Datasets are cleaned, merged, and transformed using Python. Scripts are developed in VS Code with Codex support.

PythonVS Code

AI-Assisted Analysis

Initial ideas are written as detailed prompts and refined through AI-assisted discussions to structure analysis and explore variable relationships.

AI Prompting

Visualization

Visual outputs are created using Datawrapper, Tableau, Flourish, and Matplotlib, depending on the analysis type.

DatawrapperTableauFlourish

Web Integration

Final results are integrated into the website using GitHub Pages and Lovable.dev, focusing on clarity and storytelling.

GitHubLovable

Update Process

Data and visuals can be updated by re-running Python scripts and regenerating charts. Most steps are automated with minor manual adjustments.
